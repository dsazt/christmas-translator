# christmas-translator
Интерактивный переводчик для изучения английских слов

## 🚀 Возможности
- Клик по карточке для перевода
- Перемешивание слов для запоминания
- Автосохранение прогресса

## 🛠 Установка и запуск
```bash
# Клонировать репозиторий
git clone https://github.com/ваш-username/christmas-translator.git
cd christmas-translator

# Установить зависимости
pip install -r requirements.txt

# Запустить
python app.py
